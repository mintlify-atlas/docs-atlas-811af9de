# Xtreme Fitness

Xtreme Fitness is a modern fitness web application built with React and Express/MongoDB. It serves three distinct user roles — guests, members, and admins — each with a tailored experience.

## What is Xtreme Fitness?

Xtreme Fitness is a full-stack fitness platform that combines:

- **Public marketing site** — Home, About, Services, Trainers, Pricing, Blog, and Contact pages
- **Member portal** — Authenticated members can browse and enroll in workouts
- **Admin dashboard** — Admins manage blog posts, review customer messages, and moderate reviews
- **REST API** — Express backend with MongoDB for data persistence and JWT for authentication

## User roles

The application supports three roles with distinct permissions:

| Role | Access |
|------|--------|
| **Guest** | Public marketing pages, blog, contact form |
| **Member** | All public pages + personal workout portal (`/min-side`) |
| **Admin** | All pages + full admin dashboard (`/admin`) |
