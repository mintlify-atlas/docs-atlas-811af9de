This guide walks you through cloning the repo, installing dependencies, seeding the database, and launching both the frontend and backend.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** (v18 or later recommended)
- **npm** (bundled with Node.js)
- **MongoDB** running locally

<Warning>
  The backend requires a running MongoDB instance. Make sure MongoDB is started before seeding the database or launching the server.
</Warning>

## Setup

<Steps>

  <Step title="Clone the repository">
    Clone the project from GitHub and navigate into the project root.

    ```bash
    git clone https://github.com/EmpathyGrowth/xtreme-fitness.git
    cd xtreme-fitness
    ```
  </Step>

  <Step title="Install frontend dependencies">
    From the project root, install the React frontend dependencies.

    <CodeGroup>

    ```bash npm
    npm install
    ```

    ```bash yarn
    yarn install
    ```

    ```bash pnpm
    pnpm install
    ```

    </CodeGroup>
  </Step>

  <Step title="Install backend dependencies">
    Navigate into the `backend` directory and install the Express server dependencies.

    <CodeGroup>

    ```bash npm
    cd backend
    npm install
    ```

    ```bash yarn
    cd backend
    yarn install
    ```

    ```bash pnpm
    cd backend
    pnpm install
    ```

    </CodeGroup>
  </Step>

  <Step title="Seed the database">
    From inside the `backend` directory, run the seed script to populate MongoDB with initial data.

    ```bash
    npm run "Opret Database"
    ```

    <Note>
      This command runs `node -r dotenv/config ./lib/db/mcd/mcd.js` and creates all required collections including users, blog posts, services, workouts, and more.
    </Note>
  </Step>

  <Step title="Configure the environment">
    The backend reads configuration from a `.env.local` file in the `backend` directory. Create the file if it does not exist.

    ```bash backend/.env.local
    SERVER_PORT=3042
    ```

    <Tip>
      `SERVER_PORT` defaults to `3042` if not set. You only need this file if you want to override the port or add other environment variables such as a custom MongoDB connection string.
    </Tip>
  </Step>

  <Step title="Start the backend">
    From the `backend` directory, start the Express API server with nodemon.

    ```bash
    npm run "Start Server"
    ```

    The API will be available at `http://localhost:3042`.
  </Step>

  <Step title="Start the frontend">
    Open a new terminal, go back to the project root, and start the Vite development server.

    ```bash
    npm run dev
    ```

    The frontend will be available at `http://localhost:5173`.
  </Step>

  <Step title="Open the app">
    Navigate to [http://localhost:5173](http://localhost:5173) in your browser. You should see the Xtreme Fitness home page.

    To verify everything is working, sign in using one of the provided test accounts.
  </Step>

</Steps>
